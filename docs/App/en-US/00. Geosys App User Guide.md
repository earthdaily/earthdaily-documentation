
# Geosys App User Guide



## Overview

  The Earth Daily web application utilizes a Home Page that provides a quick view of key analytics and multiple Modules which allow the user to deep dive on specific analytics at different aggregation levels. 


![EarthdailyApp](../images/FES_images/Home_View_1_Agro.jpg)

## Modules

The EarthDaily web application has many modules which are configurable by user.  Not all modules are available to every user.

By clicking on the Menu icon in the top left of the screen, you will see all modules available to the user.

![EarthdailyApp](../images/FES_images/Home_Module_List.jpg)
